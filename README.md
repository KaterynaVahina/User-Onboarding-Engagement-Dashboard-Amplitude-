## Amplitude Dashboard: User Onboarding & Engagement

Навчальний проект у Amplitude: аналіз поведінки користувачів у симульованому мобільному додатку.

**Що зроблено:**
- Налаштовано відстеження ключових подій (onboarding, feature usage)
- Побудовано воронки конверсії для онбордингу
- Проаналізовано retention та типові шляхи користувачів (journey maps)
- Виявлено точки відтоку — особливо на етапі ознайомлення з додатком

**Результат:**  
Значна частина користувачів припиняє взаємодію під час онбордингу. Ці інсайти можуть допомогти покращити UX і зменшити ранній відтік.
